Brain MRI Grad-CAM Visualization

Advanced explainable AI framework for medical imaging using multi-scale Grad-CAM 
with adaptive Gaussian smoothing and robust normalization techniques.

Features
- Multi-scale CAM with weighted Gaussian kernels
- Percentile-based normalization for scale invariance
- Adaptive gamma correction for enhanced visibility
- Multiple colormap support (Turbo, Magma, Jet)
- Alpha-blended overlay visualization

Tech Stack
PyTorch | OpenCV | Grad-CAM | NumPy | SciPy | Matplotlib

Applications
- Brain tumor detection
- Medical image interpretation
- AI model explainability
- Clinical decision support
